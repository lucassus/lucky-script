# String Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add string literals to Lucky Script — lexing, parsing, and interpretation — including `+` concatenation, `==`/`!=` comparison, boolean coercion (empty = falsy), and `display()` for the upcoming `print` feature.

**Architecture:** Full pipeline change: `StringRecognizer` handles lexing (state machine, raw value with surrounding quotes), `StringLiteral` AST node stores the raw value, and `LuckyString` runtime object decodes escapes and implements operations. Strict typing — no coercion with other types.

**Tech Stack:** TypeScript, Jest. Run tests with `yarn test`, lint with `yarn lint`, typecheck with `yarn typecheck`.

Create `src/Lexer/Recognizer/StringRecognizer.ts`:

```typescript
import { Recognizer } from "./Recognizer";
import { Case } from "./state/Case";
import { State } from "./state/State";

const beginString = new Case(0, false);

const endString: State = {
  name: 3,
  isFinal: true,
  next: (): undefined => undefined,
};

// inString is declared with definite assignment (!) to allow escape to
// reference it before the assignment below — both are always set before
// the recognizer runs.
let inString!: State;

const escape: State = {
  name: 2,
  isFinal: false,
  next(symbol: string): State | undefined {
    if (symbol === '"' || symbol === '\\' || symbol === 'n') return inString;
    return undefined;
  },
};

inString = {
  name: 1,
  isFinal: false,
  next(symbol: string): State | undefined {
    if (symbol === '"') return endString;
    if (symbol === '\\') return escape;
    return this;
  },
};

beginString.on('"').switchTo(inString);

export class StringRecognizer extends Recognizer {
  constructor() {
    super(beginString);
  }
}
```

Create `src/Interpreter/objects/LuckyString.ts`:

```typescript
import { LuckyBoolean } from "./LuckyBoolean";
import { LuckyObject } from "./LuckyObject";

export class LuckyString extends LuckyObject {
  constructor(public readonly value: string) {
    super();
  }

  add(other: LuckyObject): LuckyObject {
    this.ensureLuckyString(other);
    return new LuckyString(this.value + other.value);
  }

  eq(other: LuckyObject): LuckyBoolean {
    this.ensureLuckyString(other);
    return LuckyBoolean.fromNative(this.value === other.value);
  }

  toBoolean(): LuckyBoolean {
    return LuckyBoolean.fromNative(this.value.length > 0);
  }

  display(): string {
    return this.value;
  }

  private ensureLuckyString(other: LuckyObject): asserts other is LuckyString {
    if (!(other instanceof LuckyString)) {
      this.throwIllegalOperationError();
    }
  }
}
```

private visitStringLiteral(node: StringLiteral): LuckyString {
  const raw = node.value.slice(1, -1); // strip surrounding quotes
  const decoded = raw.replace(/\\(["\\n])/g, (_, c) => (c === 'n' ? '\n' : c));
  return new LuckyString(decoded);
}
